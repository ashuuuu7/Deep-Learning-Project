# 🐱🐶 Cat vs Dog Image Classifier

A Deep Learning project that classifies images as **Cat** or **Dog** using a Convolutional Neural Network (CNN) built with TensorFlow/Keras and deployed with Streamlit.

## 🚀 Live Demo

Deployed on Render:
https://deep-learning-project-6.onrender.com

## 📌 Project Overview

This project uses a CNN model trained on Cat and Dog images to perform binary image classification. Users can upload an image through a Streamlit web interface and receive a prediction with confidence score.

## 🛠️ Tech Stack

* Python
* TensorFlow
* Keras
* NumPy
* Pillow
* Streamlit
* Render
* Git & GitHub

## ✨ Features

* Upload Cat or Dog images
* Real-time image classification
* Confidence score prediction
* Deep Learning based CNN architecture
* Web deployment using Streamlit and Render

## 🧠 Model Architecture

* Convolutional Neural Network (CNN)
* Image Size: 64 × 64
* Binary Classification
* TensorFlow / Keras Implementation

## 📊 Performance

* Training Accuracy: ~84%
* Validation Accuracy: ~83%

## ▶️ Run Locally

### Clone Repository

```bash
git clone https://github.com/ashuuu7/Deep-Learning-Project.git
cd Deep-Learning-Project
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```

## 📸 Application Workflow

1. Upload an image
2. Image preprocessing
3. CNN prediction
4. Confidence score generation
5. Display result (Cat or Dog)
